# 📂 `src/` - Reusable Kafka Components

## 📌 Overview

The `src/` directory contains **standalone, reusable components** that can be used across multiple Kafka projects. These include **Kafka producers, consumers, Streams applications, and Kafka Connect connectors**.

## 📁 Directory Structure

Stores standalone, reusable Kafka-related code that can be used across multiple projects. These are modular components, such as producers, consumers, Kafka Streams apps, and Kafka Connect connectors.
